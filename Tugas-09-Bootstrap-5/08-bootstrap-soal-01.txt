1. Apa itu CSS Framework?
CSS Framework sendiri merupakan kumpulan kode CSS yang dapat digunakan untuk mengembangkan website. CSS Framework adalah hasil formula codingan website yang telah siap digunakan. Sehingga programmer tidak perlu menghabiskan banyak waktu dan tenaga untuk membuat formula CSS website baru dari awal

2. Apakah bootstrap satu-satunya CSS Framework yang dapat kita gunakan?
Tidak

3. Sebutkan beberapa CSS Framework selain Bootstrap
Tailwind, Materialize, SemanticUI, Foundation, Bulma, dll

4. Bagaimana tata cara penggunaan Bootstrap?
- offline : 
    1) mengunduh Bootstrap untuk mendapatkan CSS dan JavaScript yang dikompilasi,
    2) lalu unzip,
    3) lalu hubungkan file html dengan file css js yang telah diunzip,
    4) panggil class yang tersedia untuk membentuk html sesuai keinginan
- online :
    1) copy link cdn jsDeliver pada file html, <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.0.2/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-EVSTQN3/azprG1Anm3QDgpJLIm9Nao0Yz1ztcQTwFspd3yD65VohhpuuCOmLASjC" crossorigin="anonymous">
    2) panggil class yang tersedia untuk membentuk html sesuai keinginan
